# Rigol GUI

GUI for rigoal wave generator (series 4000~5000).

![image-20230425133314118](README.assets/overview.png)

Run `python start_gui.py` to launch the program.



# Dependency

```
numpy
numba
scipy
PyQt5==5.15.9
PyVISA==1.11.0
pyqtgraph==0.12.3
```
